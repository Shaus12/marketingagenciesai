# Meta Ads Creative Rules — Evidence-Based

These rules are based on research, A/B test data, and platform benchmarks (2024-2026).
Follow these for EVERY ad. No exceptions.

---

## Quality Gate (before ANY ad goes live)

1. **Typo check** — Read every word in the generated image character by character. AI generators frequently misspell words. If ANY typo: regenerate. NEVER push a typo.
2. **Spacing check** — Verify no words are jammed together (e.g., "LiveQ&A" → "Live Q&A").
3. **URL check** — Primary text ends cleanly. CTA button and link handle the click.
4. **CTA check** — Must be `SIGN_UP` for webinar ads. Link: `https://yourdomain.com/webinar`.
5. **Price check** — No product prices anywhere (webinar funnel rule).
6. **Brand check** — Image uses brand template background.
7. **User approval** — Show generated images to the user. Wait for approval. NEVER auto-push.

## Image Creative (60-80% of ad performance)

| Rule | Detail | Source |
|------|--------|--------|
| Resolution | Minimum 1080x1080. Upload max quality, let Meta compress. | Meta docs |
| Aspect ratio | **4:5 vertical (1080x1350)** is best for mobile feed — takes most screen space. 1:1 square is universal fallback. 9:16 for Stories/Reels. | 70% of tests show vertical wins |
| Text overlay | Keep under 20% of image area. Meta no longer rejects text-heavy images but still deprioritizes them in auction. | Meta policy change Sept 2021 |
| Contrast | Minimum 4.5:1 ratio between text and background. High-contrast ads boost CTR up to 32%. | LeadEnforce study |
| Dark mode | Our dark backgrounds + lime text work perfectly for dark-mode feeds. Avoid pure white backgrounds — they flash in dark mode. | AlmostZero research |
| Format | Use PNG for text-heavy graphics (sharper edges). JPG for photos. Keep under 1MB. | Meta specs |
| Refresh | Static images fatigue in 14-21 days for B2B/services. Changing only text overlay doesn't count — Meta's visual recognition treats it as the same image. | Meta, Jon Loomer |
| Scroll stop | You have **1.5 seconds** to stop the scroll. The image IS the hook. Must create curiosity or relevance instantly. | 2024 scroll speed data |

## Primary Text (15-20% of performance)

| Rule | Detail | Source |
|------|--------|--------|
| Truncation | Mobile shows **first 125 characters** before "See more." 80% never click "See more." Put your hook in the first 125 chars. | Meta, GoMarble |
| Length | For complex offers (webinars, coaching): **mid-to-long copy (80-125+ words) outperforms short copy**. Short copy cost 2.4x more per lead in A/B tests. | AdEspresso $1K experiment |
| Structure | Use **PAS** format: Problem → Agitate → Solution → CTA. One thought per line. Line break every 2-3 lines. | Madgicx playbook |
| First line | This is your text hook. Must earn the next second of attention. Don't sell — create curiosity. | Metalla Digital |
| Emojis | 1-3 strategic emojis improve scannability. Checkmarks (✓), arrows (→) as bullets work well. Don't overdo it. | Aaron Zakowski (386% engagement lift with emojis in video ads) |
| URLs | **Don't put naked URLs in primary text.** The CTA button and headline already handle clicks. URLs in copy add clutter. End with a clean call-to-action line instead. | AdEspresso, Vaizle |
| Caps | Use caps sparingly for emphasis ("FREE Workshop"). Avoid ALL CAPS sentences — reads as shouting, can trigger spam filters. | Madgicx |

## Headline (10-15% of performance)

| Rule | Detail | Source |
|------|--------|--------|
| Length | **25-35 characters** max. Mobile truncates at 27 chars. 5-word headlines perform best. | AdEspresso (752,626 ads analyzed) |
| Style | Benefit-focused beats curiosity-focused for direct response. Use specific, tangible outcomes. | AdEspresso |
| Function | Headlines appear BELOW the image. They bridge the creative to the CTA button. Complete the thought, don't repeat the image text. | Meta docs |

---

## Description (2-5% of performance)

| Rule | Detail | Source |
|------|--------|--------|
| Visibility | **NOT visible on mobile Feed or Stories** — only desktop. 85% of views are mobile. Lowest priority element. | Meta docs |
| Length | Hard limit ~30 characters. Use for urgency/qualifier: "No credit card needed" or "Free — limited spots." | Vaizle |
| Content | Add new info the other fields don't have. Don't duplicate the CTA or headline. Don't put URLs here. | Meta best practices |

---

## CTA Button (5-8% of performance)

| Rule | Detail | Source |
|------|--------|--------|
| Webinar ads | Use **SIGN_UP** — 30% more email sign-ups vs. no CTA button. Matches "register" mental model. | Databox |
| Lead magnets | Use **DOWNLOAD** — lowest CPA in testing ($0.80 vs $1.21 for Learn More). | AdEspresso |
| Cold traffic | **LEARN_MORE** is safest default for low-commitment top-of-funnel. | AdEspresso |

## Hook Patterns (ranked by effectiveness)

| # | Pattern | Example | When to Use |
|---|---------|---------|-------------|
| 1 | **Pattern interrupt** | "Stop watching AI tutorials." | Cold traffic, scroll-stoppers |
| 2 | **Number/statistic** | "90% of AI builders never make money." | Cold traffic, credibility |
| 3 | **Question** | "What if your weekend project made €500/mo?" | Cold traffic, curiosity |
| 4 | **Controversial** | "Your AI tutorials are worthless." | Cold traffic, bold angles |
| 5 | **Before/After** | "From demos → deployed products in 30 days" | Warm traffic, transformation |
| 6 | **FOMO/Urgency** | "8 days left. 47 spots remaining." | Retargeting, time-sensitive |
| 7 | **Disqualifier** | "This is NOT for everyone..." | Qualifying leads, filtering |

## Cold vs Warm Creative

| Element | Cold Traffic | Warm/Retargeting |
|---------|-------------|-----------------|
| Goal | Stop the scroll, create awareness | Remind, address objections, create urgency |
| Hooks | Stat hooks, questions, pattern interrupts | "You checked it out," "Still deciding?", urgency |
| Proof | Aggregated numbers (€XXX+, X,XXX+ builders) | Specific testimonials, detailed results |
| Copy | Educate: explain the 3-path system | Remind: list what they get, address why they didn't sign up |
| CTA | "Save your free spot" | "Register now — seats filling up" |
| Refresh | Every 14-21 days | Every 7-14 days (fatigue faster) |

## Webinar-Specific Rules

| Rule | Detail | Source |
|------|--------|--------|
| Timing | Promote no more than 10 days before the event. Too much lead time → people forget. | BeKnown Online |
| "FREE" | Use the word prominently. Lowers barrier. "Exclusive" creates belonging. | Webinar ad benchmarks |
| Countdown | Use countdown urgency in final 48 hours. | Best practice |
| CPL benchmarks | Cold: €3-8 (target €3-5). Optimized: €1-2. | WordStream 2025 |
| CTR target | Minimum 1.5%. Optimized: 3%+. | WordStream |
| Volume | Test 30 ad variations across 10 pain points. Keep top 10 by conversion rate after 1 week. | Hormozi framework |
| Budget scaling | Maximum 10% per day increase. Larger jumps reset learning phase. | Meta docs |

## Testing Framework

- **3-2-2 minimum**: 3 creatives × 2 primary texts × 2 headlines = 12 combinations
- **Minimum test duration**: 48-72 hours
- **Minimum impressions**: 1,500+ per variant before making decisions
- **Statistical confidence**: 95% target
- **500 impressions is NOT enough** for any creative decision
- Advertisers testing 3-10 creatives see cost-per-result drop up to **46%**

---

## Performance Hierarchy (what to optimize first)

```
1. Creative (image/video)     → 60-80% of performance
2. Primary text (first line)  → 15-20%
3. Headline                   → 10-15%
4. CTA button                 → 5-8%
5. Description                → 2-5%
```

Spend 80% of creative effort on the visual and primary text hook. Everything else is fine-tuning.
